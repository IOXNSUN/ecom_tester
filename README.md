# ecom_tester

![Главный скриншот](screenshots/screen1.jpg)

## О проекте
Автоматическое тестирование e-commerce процессов: заказы, платежи, проверки статусов.

## Технологии
- Python 3.10
- Requests, asyncio
- Postman API / тестовые скрипты
